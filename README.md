# Multi Timer Web Application

![Status](https://img.shields.io/badge/status-completed-green)   ![Frontend](https://img.shields.io/badge/frontend-JavaScript-blue)   ![UI](https://img.shields.io/badge/UI-custom-purple)

A web-based time management application that combines **clock, timer, and stopwatch functionality** in a single interface.

The app is designed with a focus on **clean UI, usability, and interactive user experience**, allowing users to track time efficiently in different scenarios.

---

## 🚀 Features

- Real-time clock
- Stopwatch (start / stop / reset)
- Timer functionality
- Ability to manage multiple timers
- Independent timer control
- Clean and user-friendly UI

---
## 🛠 Tech Stack

- **HTML5**
- **CSS3** (Flexbox, Grid)
- **JavaScript (ES6+)**

---
